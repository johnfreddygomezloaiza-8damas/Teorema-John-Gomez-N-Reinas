# Teorema-John-Gomez-N-Reinas
Con este teorema facilito la exploración del problema de las 8 damas y el recorrido del caballo, optimizando el tiempo de manera sustancial para cálculos grandes que normalmente toman mucho tiempo, generando una herramienta practica para agilizar los procesos computacionales complejos y de gran tamaño
